# SensitiveDisplay 敏感信息显示

用于安全显示敏感信息的组件，支持隐藏/显示切换和复制功能。

## 何时使用

- 需要显示敏感信息（如 API 密钥、访问令牌）
- 需要隐藏/显示切换功能
- 需要支持敏感信息的复制功能
- 用户配置、系统设置等场景

## API

SensitiveDisplay 基于 Ant Design Typography 封装。

### 扩展属性

| 参数 | 说明 | 类型 | 默认值 | 必填 |
| --- | --- | --- | --- | --- |
| text | 敏感信息文本 | `string` | - | ✅ |

## 组件特点

1. **安全显示** → 默认隐藏敏感信息，点击切换显示/隐藏状态
2. **一键复制** → 基于 Ant Design Typography 的复制功能
3. **自动省略** → 文本过长时自动显示省略号（最大宽度 120px）
4. **国际化支持** → 提示信息支持国际化配置

## 显示状态

- **隐藏状态**：显示加密图标（TokenFilled），不显示实际内容
- **显示状态**：显示实际文本内容，支持省略号处理
- **切换方式**：点击图标或文本区域进行状态切换

## 注意事项

1. 组件需要包裹在 `ConfigProvider` 中确保主题正常工作
2. 文本过长时会自动显示省略号（最大宽度 120px）
3. 组件会自动阻止事件冒泡，避免影响父组件事件处理
4. 复制功能基于 Ant Design Typography 组件实现

## 最佳实践

1. **API 密钥**：用于显示 API 密钥、访问令牌等敏感数据
2. **密码字段**：配置页面中的密码、密钥等信息展示
3. **用户隐私**：显示用户隐私信息时，默认隐藏，用户主动查看
4. **复制提示**：建议配合 Tooltip 或 Message 提示用户复制成功
