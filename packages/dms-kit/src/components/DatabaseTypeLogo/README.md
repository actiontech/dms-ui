# DatabaseTypeLogo 数据库类型图标

用于显示数据库类型的图标和名称，提供统一的视觉识别。

## 何时使用

- 需要在界面中展示数据库类型标识
- 数据库列表、表格或卡片中显示数据库信息
- 数据库连接配置和选择场景

## API

### DatabaseTypeLogoProps

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| dbType | 数据库类型名称 | `string` | - |
| logoUrl | 数据库图标地址 | `string` | - |

## 组件特点

- **统一样式**：固定的图标大小（16px）和间距（4px）
- **简洁布局**：图标+文字的横向排列，自动处理溢出
- **文本提示**：鼠标悬停显示完整数据库类型名称

## 注意事项

- 两个参数 `dbType` 和 `logoUrl` 都是必需的
- 图标会自动适配高度，保持宽度为 16px
- 组件使用 `white-space: nowrap` 和 `overflow: hidden`，注意父容器宽度

## 最佳实践

- 确保 `logoUrl` 指向有效的图标资源
- 在表格或列表中使用时，考虑设置合适的列宽以完整显示内容
