# ReminderInformation 提醒信息

基于 Ant Design Typography 封装，提供统一的状态反馈提示。

## 何时使用

- 需要显示操作成功或失败的反馈信息
- 需要图标和文字组合的状态提示
- 需要简洁明了的状态反馈

## API

ReminderInformation 基于 Ant Design Typography 封装。

| 参数 | 说明 | 类型 | 默认值 | 必填 |
| --- | --- | --- | --- | --- |
| status | 状态类型 | `'success' \| 'error'` | - | ✅ |
| message | 提醒文本内容 | `string` | - | ✅ |

## 组件特点

1. **统一样式规范** → 自动应用项目主题系统的样式
2. **状态反馈** → 支持 success 和 error 两种状态
3. **图标配合** → 状态对应的图标自动显示
4. **自动对齐** → 图标和文字垂直居中对齐

## 状态说明

- **success**：绿色主题，显示成功图标
- **error**：红色主题，显示错误图标

## 注意事项

1. 组件需要包裹在 `ConfigProvider` 中确保主题正常工作
2. `status` 只支持 `'success'` 和 `'error'` 两种值
3. 建议控制 `message` 长度保持简洁，长文本会自动换行

## 最佳实践

1. **消息简洁**：避免冗长的描述，突出关键信息
2. **及时反馈**：在操作完成后立即显示相应的提醒信息
3. **位置合理**：将提醒信息放置在用户容易看到的位置
4. **状态准确**：根据实际操作结果选择正确的状态类型
