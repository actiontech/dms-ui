# EmptyBox 空盒子

一个简单的条件渲染容器组件，根据布尔值决定是否渲染子组件或默认节点。

## 何时使用

- 需要根据条件控制组件的显示/隐藏
- 需要在条件不满足时显示替代内容

## API

### EmptyBoxProps

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| if | 是否渲染子组件 | `boolean` | `true` |
| children | 条件为 true 时渲染的内容 | `ReactNode` | - |
| defaultNode | 条件为 false 时渲染的内容 | `ReactElement \| JSX.Element \| string` | `null` |

## 组件特点

- **简单高效**：纯逻辑组件，无额外渲染开销
- **条件渲染**：条件为 false 时完全不渲染子组件，节省资源
- **默认节点**：支持在条件不满足时显示替代内容
- **即时响应**：条件变化时立即更新渲染状态

## 注意事项

- 条件变化时子组件会被完全卸载和重新挂载，状态会重置
- 频繁切换条件可能导致性能问题，建议合理使用

## 最佳实践

- 对于需要保留状态的场景，使用 CSS `display` 控制显示/隐藏而非条件渲染
- 结合 `defaultNode` 提供良好的加载或空状态提示
