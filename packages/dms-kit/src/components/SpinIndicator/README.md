# SpinIndicator 加载指示器

基于 SVG 动画的自定义加载指示器组件，提供了统一的样式规范和流畅的动画效果，支持自定义尺寸和样式类名。

## 何时使用

- 需要自定义加载动画时
- 需要替代默认的 Ant Design Spin 组件时
- 需要特定样式的加载指示器时
- 需要保持与设计系统一致的加载组件样式时
- 需要轻量级的加载动画时

## API

### SpinIndicator

| 参数 | 说明 | 类型 | 默认值 | 版本 |
| --- | --- | --- | --- | --- |
| width | 指示器宽度 | `number` | `24` | - |
| height | 指示器高度 | `number` | `30` | - |
| className | 自定义类名 | `string` | - | - |

## 设计规范

### 样式特性

- 基于 SVG 的矢量图形
- 流畅的 CSS 动画效果
- 支持自定义尺寸
- 继承当前文本颜色
- 响应式设计支持

### 动画规范

- **动画时长**: 0.6 秒循环
- **动画类型**: 透明度、高度、位置变化
- **动画延迟**: 三个矩形分别延迟 0s、0.15s、0.3s
- **循环方式**: 无限循环

### 尺寸规范

- **默认尺寸**: `24px × 30px`
- **最小尺寸**: 建议不小于 `16px × 20px`
- **最大尺寸**: 无限制，但建议不超过 `64px × 80px`
- **宽高比**: 固定为 `4:5`

### 主题配置

组件样式通过主题系统进行配置，支持以下主题变量：

```typescript
theme.sharedTheme.components.spinIndicator = {
  // 继承 CommonIconStyleWrapper 的主题配置
}
```

## 功能特性

### 动画效果

组件使用三个矩形元素创建波浪式动画效果：
1. **第一个矩形**: 无延迟，从左侧开始动画
2. **第二个矩形**: 延迟 0.15 秒，从中间开始动画
3. **第三个矩形**: 延迟 0.3 秒，从右侧开始动画

### 自定义尺寸

- 支持任意数值的宽度和高度
- 自动保持 4:5 的宽高比例
- 响应式适配不同屏幕尺寸
- 支持像素和相对单位

### 样式继承

- 自动继承父元素的文本颜色
- 支持通过 className 自定义样式
- 集成 CommonIconStyleWrapper 样式系统
- 支持 CSS 变量和主题配置

## 使用场景

### 页面加载状态

在页面或组件加载时，显示自定义的加载动画。

### 数据加载指示

在数据请求过程中，提供友好的加载反馈。

### 操作状态反馈

在用户操作执行过程中，显示操作状态。

### 替代默认加载器

当需要特定样式的加载指示器时，替代 Ant Design 的默认组件。

## 注意事项

1. 组件需要包裹在 `ConfigProvider` 中以确保主题正常工作
2. 尺寸参数会影响动画效果，建议保持合理的比例
3. 组件使用 SVG 动画，在低端设备上可能有性能影响
4. 自定义样式时注意不要破坏动画效果
5. 组件依赖 CommonIconStyleWrapper 样式系统

## 最佳实践

1. **尺寸选择**: 根据使用场景选择合适的尺寸，避免过大或过小
2. **样式定制**: 通过 className 进行样式定制，保持动画效果
3. **性能考虑**: 在大量使用时要考虑性能影响
4. **主题集成**: 利用主题系统保持样式一致性
5. **响应式设计**: 在不同屏幕尺寸下测试显示效果
