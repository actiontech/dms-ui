# SpinIndicator 加载指示器

基于 SVG 动画的加载指示器组件，提供流畅的波浪式动画效果，支持自定义尺寸和颜色。

## 何时使用

- 页面或组件数据加载时
- 按钮操作等待反馈时
- 需要轻量级加载动画时

## API

| 参数 | 说明 | 类型 | 默认值 | 必填 |
| --- | --- | --- | --- | --- |
| width | 指示器宽度（像素） | `number` | `24` | - |
| height | 指示器高度（像素） | `number` | `30` | - |
| className | 自定义类名 | `string` | - | - |

## 组件特点

1. **波浪式动画** → 三个矩形依次动画，形成流畅的波浪效果
2. **自动继承颜色** → 继承父元素的文本颜色，无需额外配置
3. **灵活尺寸** → 支持自定义宽高，默认 24×30px
4. **轻量级** → 基于 SVG，性能优秀，适合大量使用

## 注意事项

1. 必须在 `ConfigProvider` 中使用
2. 建议尺寸保持 4:5 的宽高比（如 24×30、16×20、32×40）
3. 通过父元素的 `color` 样式控制颜色
4. 避免过大尺寸（建议不超过 64×80px）

## 最佳实践

1. **按钮中使用**：设置较小尺寸（如 16×20px）配合文字
2. **列表加载**：使用默认尺寸（24×30px）居中显示
3. **页面级加载**：使用大尺寸（48×60px 或 64×80px）配合提示文字
4. **颜色控制**：通过父容器的 `style={{ color: '#1890ff' }}` 设置颜色
